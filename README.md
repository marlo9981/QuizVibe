
# 💕 Love Language Quiz - Valentine's Edition

Interactive 30-question quiz based on Gary Chapman's 5 Love Languages.

## Features
- Forced-choice questions (never repeat)
- Instant animated scoring
- Responsive Valentine's theme
- Pure HTML/CSS/JS - no backend

## Deploy in 2 mins
1. Fork this repo or upload `index.html`
2. Vercel: Import → Deploy (free)
3. GitHub Pages: Settings → Pages → Deploy from branch

Live demo: [your-vercel-link]

## Customization
Edit `questions` array in `<script>` section.
